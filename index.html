🐍 Modelos de Datos en Python (Clases)

La arquitectura del backend mapea perfectamente estas entidades lógicas para la base de datos relacional. 

class Usuario:
    id_usuario: int
    nombre: str
    telefono: str
    geolocalizacion: str
    nivel_de_juego: str

class ComplejoDeportivo:
    id_complejo: int
    nombre: str
    direccion: str
    coordenadas: dict # {"lat": float, "lng": float}
    horarios_atencion: str

class Cancha:
    id_cancha: int
    id_complejo: int # Clave foránea
    tipo_deporte: str
    precio_por_hora: float
    tiene_techado: bool
    tiene_luz: bool

class Reserva:
    id_reserva: int
    id_usuario: int  # Clave foránea
    id_cancha: int   # Clave foránea
    fecha: str
    hora_inicio: str
    hora_fin: str
    estado: str  # "Pendiente" | "Pagado" | "Cancelado"
